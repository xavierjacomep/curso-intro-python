# 🐍 Python para Economía y Administración de Empresas

Curso introductorio de Python orientado a estudiantes de Economía y Administración **sin experiencia previa en programación**. Los ejemplos y ejercicios usan contexto empresarial latinoamericano: ventas, inventarios, cooperativas de ahorro, empresas ecuatorianas, y más.

**Herramienta recomendada:** [Google Colab](https://colab.research.google.com/) — no requiere instalación, corre directo en el navegador.

## ▶️ Cómo usar los notebooks en Colab

1. Haz clic en el botón **Open in Colab** del módulo que quieras abrir.
2. En Colab, ve a **Archivo → Guardar una copia en Drive** para conservar tus cambios.
3. Ejecuta cada celda con **Shift + Enter**.

> Los notebooks están listos para correr sin instalación adicional, salvo el módulo 14 que instala `pandas` y `openpyxl` automáticamente en la primera celda.
